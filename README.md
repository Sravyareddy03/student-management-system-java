# Student Management System (Java)

## 📌 Project Description

This is a simple console-based Student Management System built using Java.
It allows users to perform basic operations like adding, deleting, searching, and viewing student records.

---

## 🚀 Features

* Add a new student
* Delete a student by ID
* Search for a student
* Display all students
* Menu-driven interface

---

## 🛠️ Technologies Used

* Java
* OOP (Object-Oriented Programming)
* ArrayList

---

## 🧠 Concepts Implemented

* Classes and Objects
* Encapsulation
* ArrayList operations
* Looping and conditional statements
* User input handling using Scanner

---

## ▶️ How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/Sravyareddy03/student-management-system-java.git
   ```

2. Navigate to the folder:

   ```bash
   cd student-management-system-java
   ```

3. Compile the program:

   ```bash
   javac Mainclass.java
   ```

4. Run the program:

   ```bash
   java Mainclass
   ```
